# Решение задачи X о строительстве газового трубопровода

В постановке задачи требуется решение со сложностью  O(N) по времени

Приведены варианты решения с помощью рекурсивного поиска медианы целых неотсортированных чисел 
и с помощью сортировки массива быстрой сортировкой с последующим поиском медианы.

# Тесты

Для запуска тестов используется команда из корня проекта:

```bash
 ./vendor/bin/phpunit Tests
```

phpunit устанавливается с помощью composer
```
composer install
```
